# Kronos Stock Predictor

AI-powered stock prediction using the Kronos time-series foundation model with ACWI benchmark comparison.

## Features

- **20-day price prediction**: Uses 40-day lookback with Kronos-base model
- **ACWI benchmark**: Global market comparison with cached predictions
- **Technical statistics**: SMA, EMA, RSI, MACD, Bollinger Bands, ATR, Volume
- **Interactive charts**: 1-year % return vs ACWI, candlestick predictions
- **Confidence bands**: Prediction uncertainty visualization

## Setup

### Requirements

- Python 3.11+
- 2GB+ RAM for model loading

### Install

```bash
pip install -r requirements.txt
```

### Run Locally

```bash
streamlit run app.py
```

## Deployment

### Railway

1. Fork this repository
2. Create a new project on [Railway](https://railway.app)
3. Select "Deploy from GitHub repo"
4. Railway auto-detects the Dockerfile
5. Deploy (~$5-10/mo for 2GB RAM)

### Docker

```bash
docker build -t kronos-web .
docker run -p 8501:8501 kronos-web
```

## Important Notes

- First launch downloads ~~410MB Kronos model weights from HuggingFace (~~30s)
- ACWI predictions are cached globally after first run
- All predictions are generated by a general time-series model, not financial advice

## Model Architecture

Uses Kronos-base via `NeoQuasar/Kronos-Tokenizer-base` and `NeoQuasar/Kronos-base` model weights (HuggingFace Hub). Model architecture is based on a transformer with VQ-tokenization for time-series forecasting.
