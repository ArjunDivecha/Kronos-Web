# Kronos Stock Predictor

AI-powered stock prediction using the Kronos time-series foundation model, compared against a selectable ETF benchmark.

## Features

- **Configurable horizons**: Sidebar sliders for lookback and prediction length (defaults 40 / 20 days) with Kronos-base
- **Benchmark ETF (dropdown)**: Compare against **ACWI** (MSCI ACWI), **SPY** (S&P 500), or **CQQQ** (China tech). Each choice runs a separate cached benchmark forecast
- **Technical statistics**: SMA, EMA, RSI, MACD, Bollinger Bands, ATR, Volume
- **Interactive charts**: 1-year % return vs chosen benchmark, plus candlestick predictions
- **Confidence bands**: Prediction uncertainty visualization (`sample_count=10` in the model call)

## Setup

### Requirements

- Python 3.11+
- 2GB+ RAM for model loading

### Install

```bash
pip install -r requirements.txt
```

### Run Locally

```bash
streamlit run app.py
```

## Deployment

### Railway

1. Fork this repository
2. Create a new project on [Railway](https://railway.app)
3. Select "Deploy from GitHub repo"
4. Railway auto-detects the Dockerfile
5. Deploy (~$5-10/mo for 2GB RAM)

### Docker

```bash
docker build -t kronos-web .
docker run -p 8501:8501 kronos-web
```

## Important Notes

- First launch downloads ~410MB Kronos model weights from HuggingFace (~30s)
- Benchmark predictions are cached per ETF and per lookback/prediction settings (use **Clear benchmark cache** under Settings to reset)
- All predictions are generated by a general time-series model, not financial advice

## Model Architecture

Uses Kronos-base via `NeoQuasar/Kronos-Tokenizer-base` and `NeoQuasar/Kronos-base` model weights (HuggingFace Hub). Model architecture is based on a transformer with VQ-tokenization for time-series forecasting.
